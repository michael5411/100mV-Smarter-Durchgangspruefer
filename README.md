# 100mV-Smarter-Durchgangspruefer
Dieses Gerät testet Durchgang und Widerstände bis 100Ω nur mit 100 mV. Dadurch können Verbindungen und viele Komponenten in eingebautem Zustand gecheckt werden, ohne dass Halbleiterübergänge die Messungen verfälschen. Das Messgerät gibt verschiedene Tonfolgen in unterschiedlichen Widerstandsbereichen aus. Es ist programmierbar.
